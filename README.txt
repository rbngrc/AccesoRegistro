NOTAS GENERALES
---------------

Se trata de una aplicación de login con respaldo en una base de datos la cual usa Hibernates para realizar las operaciones "CRUD".

La aplicación te da 3 intentos por persona, tras lo cual el usuario se bloquea y tiene que contactar con el administrador para su desbloqueo
El administrador también dispone de 3 intentos. Su usuario es "Admin" y la contraseña es "Admin".

Se adjunta base de datos recomendable importar, en ella se encuentran el usuario "Admin" y los intentos de cada usuario que tienen para acceder.

OTRAS CONSIDERACIONES
---------------------

Se recomienda esperar unos segundos antes de elegir provincia, la lista tarda unos segundos en cargar.
La lista no se muestra ordenada.
Se debe introducir dos dígitos, por ejemplo 02, o 47, para que la aplicación reconozca la provincia seleccionada.
